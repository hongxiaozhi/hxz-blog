# Release Plan: v1.5

最后更新：2026-03-31
状态：进行中

## 版本目标

统一全站交互反馈与文案风格，让列表、详情、评论和编辑页的状态提示更一致、更可信。

## 用户可见变化

- loading / 成功 / 失败提示采用统一样式与文案语气
- 空状态与错误状态文案更一致、更容易理解
- 页面增加轻量的使用说明与可信提示

## 范围边界

本版本包含：
- 全站提示条样式与反馈文案统一
- 列表页、详情页、评论区、编辑页空状态与错误状态统一
- 轻量的新手提示 / 可信提示

本版本不包含：
- 文章状态系统
- 图片上传
- 多用户能力

延后到后续版本：
- 文章草稿 / 归档 -> `v1.6`
- Markdown 图片上传 -> `v1.7`
