# Release Plan: v1.3

最后更新：2026-03-31
状态：已完成

## 版本目标

把博客的核心浏览与评论体验打磨到“顺手、稳定、移动端可用”。

## 用户可见变化

- 首页列表从传统分页改成“加载更多”
- 详情页信息层级更清晰，长文阅读更稳定
- 评论输入、提交、反馈流程更顺畅

## 范围边界

本版本包含：
- 首页分页与列表卡片体验优化
- 详情页节奏与评论区体验优化
- 关键移动端适配

本版本不包含：
- 文章状态系统
- 图片上传
- 多用户能力

延后到后续版本：
- 文章草稿/归档 -> `v1.6`
- 图片上传 -> `v1.7`
