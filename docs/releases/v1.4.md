# Release Plan: v1.4

最后更新：2026-03-31
状态：已完成

## 版本目标

提升编辑效率，把写作页打磨到“切换顺手、保存稳定、小屏可用”。

## 用户可见变化

- 编辑页支持双栏 / 单栏切换
- 保存入口更稳定，编辑过程中更容易触达
- 小屏设备下编辑页布局更清晰，预览不再挤压输入区

## 范围边界

本版本包含：
- 编辑页双栏 / 单栏布局
- 保存操作的显性入口与状态反馈
- 编辑页关键移动端适配

本版本不包含：
- 文章状态系统
- 图片上传
- 多用户能力

延后到后续版本：
- 文章草稿 / 归档 -> `v1.6`
- Markdown 图片上传 -> `v1.7`
