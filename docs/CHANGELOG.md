# Changelog

## v1.3.0 - 2026-03-31

### Added
- 首页列表状态摘要、空状态说明与“已全部加载”提示
- 详情页标题区、文章元信息区与评论刷新入口
- 评论加载失败重试与成功态提示

### Changed
- 详情页长文阅读区域、代码块与表格在移动端的展示更稳定
- 评论输入区改为更完整的展开式交互，补齐提交中与取消状态
- API 针对文章不存在、评论不存在、未登录与登录失效场景统一返回文案

### Fixed
- 列表分页参数在异常输入下返回不稳定的问题
- 评论区加载失败后缺少明确反馈的问题
- 小屏设备下头部、详情页与评论区按钮容易拥挤的问题

## v1.2.0 - 2026-03-29

### Added
- 评论限流能力
- 列表排序能力

### Changed
- 排序控件从占位状态切换为可用状态
- 列表查询接口支持新的排序参数

### Fixed
- 评论高频提交缺少保护的问题
- 排序交互与后端返回不一致的问题

### Notes
- 当前部署方式已逐步切换到根目录统一 Docker 编排

## v1.1.0 - 2026-03-28

### Added
- bcrypt 密码校验
- JWT 密钥环境变量化
- 评论删除接口与前端删除能力

### Changed
- 登录表单不再预填默认密码
- 安全相关配置从硬编码调整为环境变量注入

### Fixed
- 明文密码暴露风险
- 评论缺少管理入口的问题
